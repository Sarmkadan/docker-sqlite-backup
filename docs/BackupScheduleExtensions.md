# BackupScheduleExtensions

Provides extension methods for `BackupSchedule` instances to compute next run times, determine whether a backup should be performed, and generate standardized backup names and file extensions.

## API

### `GetNextRunTime`
Determines the next scheduled run time for a backup based on the schedule's recurrence pattern.

- **Parameters**
  - `schedule` (`BackupSchedule`): The schedule to evaluate.
  - `now` (`DateTime`): The current reference time for calculating the next run.
- **Return value**
  Returns a `DateTime?` representing the next scheduled run time, or `null` if the schedule does not recur or has no future runs.
- **Exceptions**
  Throws `ArgumentNullException` if `schedule` or `now` is `null`.

---

### `ShouldPerformBackup`
Indicates whether a backup should be performed at the given time according to the schedule.

- **Parameters**
  - `schedule` (`BackupSchedule`): The schedule to evaluate.
  - `time` (`DateTime`): The time to check against the schedule.
- **Return value**
  Returns `true` if a backup should be performed at the given time; otherwise, `false`.
- **Exceptions**
  Throws `ArgumentNullException` if `schedule` is `null`.

---

### `GetBackupName`
Generates a standardized name for a backup based on the schedule and the current time.

- **Parameters**
  - `schedule` (`BackupSchedule`): The schedule associated with the backup.
  - `time` (`DateTime`): The time at which the backup is being performed.
- **Return value**
  Returns a `string` formatted as `"<schedule.Name>_<time:yyyyMMddHHmmss>"`.
- **Exceptions**
  Throws `ArgumentNullException` if `schedule` or `time` is `null`.

---
### `GetBackupFileExtension`
Returns the file extension for backup files generated by the schedule.

- **Parameters**
  - `schedule` (`BackupSchedule`): The schedule associated with the backup.
- **Return value**
  Returns a `string` representing the file extension (e.g., `".dbbackup"`).
- **Exceptions**
  Throws `ArgumentNullException` if `schedule` is `null`.
