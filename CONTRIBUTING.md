# Contributing to docker-sqlite-backup

Thank you for your interest in contributing to docker-sqlite-backup!

## How to contribute
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Make your changes, following the existing coding style (see `.editorconfig`).
4. Ensure tests pass (`dotnet test`).
5. Open a pull request against the `main` branch.

## Code of Conduct
Please be respectful and constructive in all interactions.

## Reporting Issues
Please use the provided issue templates for bug reports or feature requests.
